## Technologies used ### Preact <img style="width:50px;" src="https://cdn.auth0.com/blog/preact/logo.png"> Preact is a JavaScript library that offers a fast 3kb alternative to React with the same ES6 API. Currently, Preact has over 10,000 stars on GitHub.
